Pau Desumvila del Burgo / 2019

PROGRAMA OBTENCIO D'INFORMACIO D'ARXIUS DE TEXT PLA

Petit programa per extreure informacio a partir d'arxius en format text pla. El programa treballa sempre
sobre un directori a l'escriptori anomenat AbstractTool. Si aquest directori no existeix, el programa el 
creara. Tot seguit ens donara l'opcio d'afegir els arxius en format de text pla que volem analitzar. 
A mes a mes, el programa ens crea un arxiu de text pla dintre del directori abans mencionat on es troben 
les paraules que volem eliminar. Podem afegir paraules en aquest fitxer.

A partir d'un fitxer de text pla obtindrem:

- Un arxiu de nom "nom_fitxer_original_info.txt".
- Un arxiu amb les paraules que volem eliminar.
- Un fitxer en format XML que contindra totes les paraules del text acompanyades del numero d'ocurrencies.

El fitxer "nom_fitxer_original_info.txt" contindra:

- El nom del fitxer original.
- La extensio del fitxer original.
- La data en la qual hem realitzat l'extraccio d'informacio.
- El numero de paraules que conte el fitxer original.
- Tematica: Les 5 paraules amb mes ocurrencies al fitxer original.

Dintre del projecte podeu trobar un arxiu anomenat ArticleExemple.txt que podeu fer servir per probar.
